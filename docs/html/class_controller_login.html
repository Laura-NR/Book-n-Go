connexion ( )

Contrôle la connexion d'un utilisateur

Vérifie que les champs de connexion sont remplis, puis vérifie l'email et le mot de passe en appelant la methode findByMail() de la classe UserDao. Si l'utilisateur est trouvé, il est connecté et redirigé vers la page de dashboard.
