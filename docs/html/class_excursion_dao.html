◆ creer()

creer ( array $data)

Crée une nouvelle excursion dans la base de données.

Cette méthode insère les informations d'une excursion dans la table excursion. Elle retourne un objet Excursion avec les données de la nouvelle excursion, ou null en cas d'échec.

Paramètres
array$dataDonnées de l'excursion à insérer.
Renvoie
Excursion|null L'objet Excursion créé ou null si l'insertion échoue.
Exceptions
PDOExceptionSi une erreur de base de données se produit.
