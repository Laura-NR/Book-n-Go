◆ afficherDashboard()

afficherDashboard ( )

Affiche le tableau de bord en fonction du rôle de l'utilisateur connecté.

Cette méthode vérifie le rôle de l'utilisateur dans la session et affiche le tableau de bord approprié en utilisant le template 'dashboard.html.twig'. Actuellement, les guides et les voyageurs partagent la même page d'accueil de tableau de bord.
