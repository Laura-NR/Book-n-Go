◆ creer()

creer ( )

Crée un nouveau carnet de voyage.

Cette méthode traite le formulaire de création de carnet et insert le carnet dans la base de données. Si le formulaire est envoyé avec des données valides, le carnet est ajouté via la méthode "inserer" de la classe "CarnetVoyageDAO". Si le formulaire n'est pas envoyé, le formulaire de création de carnet est affiché.

Renvoie
void

◆ lister()

lister ( )

Liste tous les carnets de voyage.

Cette méthode charge le template "carnet_voyage.html.twig" et l'affiche avec les données des carnets de voyage. Les carnets sont récupérés via la méthode "findAll" de la classe "CarnetVoyageDAO".

Renvoie
void

◆ listerParVoyageur()

listerParVoyageur ( int $idVoyageur)

Lister les carnets de voyage appartenant à un voyageur.

Cette méthode charge le template "liste_carnets_dashboard.html.twig" et l'affiche avec les carnets de voyage correspondant au voyageur d'ID $idVoyageur. Les carnets sont récupérés via la méthode "findAllByIdVoyageur" de la classe "CarnetVoyageDAO".
